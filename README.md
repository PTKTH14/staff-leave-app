# ระบบลาเจ้าหน้าที่ - GitHub Pages PWA

โฟลเดอร์นี้เก็บเฉพาะหน้าเปิดระบบ ไอคอน และไฟล์ติดตั้ง PWA ไม่มีข้อมูลเจ้าหน้าที่ PIN หรือข้อมูลการลา

## เผยแพร่

1. สร้าง GitHub repository ชื่อ `staff-leave-app` แบบ Public
2. อัปโหลดไฟล์และโฟลเดอร์ทั้งหมดจากชุดนี้ไว้ที่หน้าหลักของ repository
3. เปิด Settings > Pages
4. ที่ Build and deployment เลือก Deploy from a branch
5. เลือก branch `main` และโฟลเดอร์ `/(root)` แล้วกด Save
6. รอจน GitHub แสดงลิงก์ `https://ชื่อบัญชี.github.io/staff-leave-app/`

เปิดลิงก์ที่ได้ด้วย Safari บน iPhone/iPad หรือ Chrome บน Android แล้วเพิ่มลงหน้าจอหลัก
